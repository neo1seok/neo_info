
*TITLE:
라즈베리파이 gpio 사용 python

*ISSUE:
라즈베리파이 에서 python wiringPi모듈을 사용해서 gpio 를 통해 1io 를 통해 값을 얻어 오는 과정에서 python에서 직접 wiringPi모듈을 얻어 오는 것을 코딩할 경우 
delaytime 을 주는 과정등에서 c++모듈과 다르므로 적확한 값을 얻어오지 못한다.

아마 python자체가 스크립트 언어라서 라즈베리파이에서는 성능 차이가 난다. 일반적인 동작은 상관 없으나. 딜레이 타임 등의 민감한 부분에서 차이가 나므로 이렇게 접근하는 것은 문제가 많아 보인다.


*SOLUTION:
c++라이브러리를 python에서 읽어오는 방법으로 해결
그리고 GPIO 모듈을 이용해서 정상적으로 동작 하는 케이스도 보았다.
